Claude Code Toolkit

A desktop-app-first toolkit that keeps Claude sharp on long builds — a bounded-build workflow, auto-advance, in-session review, and safety hooks. Works in both surfaces: the Code tab (plugin) and Cowork (skills).

Install (Code tab, no terminal): open the Code tab → + → Plugins → Add plugin → add justinw916-sketch/SuperClaudePublic → install claude-code-toolkit → restart. Full steps below.
